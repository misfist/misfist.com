TILES THEME 

This is a very easy to use theme. Just create posts with a featured image. If no featured image is uploaded the 'No Image' dummy.png image located in the images folder will be displayed.
Featured images will be resized to 300px wide and unlimited height.

UPLOAD A LOGO IMAGE
You can upload your own logo image by going to 'Appearance-Customize-Theme Options'.

SOCIAL MEDIA ICONS
Go to 'Appearance-Customize-Social Media Icons' and add the url of the social media account you want to activate.

WIDGET LOCATIONS
This theme has one widget location. The Right Sidebar.
