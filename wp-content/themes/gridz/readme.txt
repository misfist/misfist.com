Installation
=============
Go to Appearance -> Themes -> Add New -> Upload. Select the theme zip folder

Theme Settings
==============
Go to Appearance -> Themes -> Theme Options. Customize the theme according to your needs

Set Image Size
===============
1. After installing the theme, first thing to do is, install "Regenerate Thumbnails" plugin from wordpress site if you already do not have it.
2. To download plugin, visit: http://wordpress.org/plugins/regenerate-thumbnails/
3. Install & activate the plugin
4. Go to Tools -> Regen Thumbnails and Click "Regnerate All Thumbnails"

Page Templates
==============
1. Fullwidth (page-fullwidth.php) -> Use this template for displaying full width page (without sidebar)
